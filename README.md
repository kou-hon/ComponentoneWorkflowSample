# ComponentoneWorkflowSample
ワークフローによるビルド、テストサンプル

## ライセンス登録の注意点
ライセンス有効可は1日9回なので、ワークフローは1日1回としてみた  
C1ビルドが必要なワークフロー、C1ビルド不要なワークフロー（テストなど）をわけて運用を考える
